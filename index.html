Featured project

Movie App

This is a react native movies app that uses TMDB API to fetch movie data. Utilizing Firebase, users are able to signup and login with an email and password and view/edit their profile details on a profile screen. Other screens include Home, Search, and Saved. The homescreen displays trending movies as well as upcoming and top rated. Clicking on a movie displays more details about the movie including a description, cast, and similar movies. The search screen allows users to search for movies and the saved screen allows users to save their favorite movies. This project really put my knowledge and skills to the test in both front-end and back-end web development. Click the link above to see a video demo of the app!
